LazyStackGetAwsSettings Console App

The LazyStackGetAwsSettings console app project is a tool that uses AWS CloudFormation to query the contents of a published AWS Stack and then generate a json file containing the information necessary for your client to connect to and use Cognito Authentication Provider provisioned in that stack.

This utility produces AwsSettings.json files as those produced by the LazyStack Visual Studio extension and CLI tools.

This tool is useful if you are using the LazyStackAuth library but not using LazyStack to generate your stacks.
