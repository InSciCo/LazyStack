Create a Xamarin Forms App

In this step we will create a Xamarin app with a fully functional Authentication process and one simple page with content produced by a call against the PetStore class.

Create the PetStoreMobileApp Project(s)

  1. Right Click on "Solution PetStore" in the Solution Explorer. The context menu appears.
  2. Select "Add → New Project". The "Add new project" dialog appears.
  3. Type "Mobile App (Xamarin.Forms)" in the search field. The project template is listed.
  4. Edit the project name to read "PetStoreMobileApp".
  5. Click on the Create button. The "New Mobile App" dialog appears.
  6. Accept defaults (Flyout, Android and iOS) and click the Create button. Three projects are created:
    • PetStoreMobileApp -- this is the Xamarin.Forms project.
    • PetStoreMobileApp.Android
    • PetStoreMobileApp.iOS

    Configure the PetStoreMobileApp Project

    1. Add the following project references:
      • LazyStackAuth
      • PetStoreClientSDK
    2. Add the following package reference:
      • Microsoft.Extensions.Configuration.Json
    3. Double Click on the PetStoreMobileApp project. The project's csproj file opens in an editor tab.
    4. Change the TargetFramework from "netstandard2.0" to "netstandard2.1".
    5. Insert the following ItemGroup into the csproj file. 
      
  <ItemGroup>
    <EmbeddedResource Include="..\..\Stacks\Dev\AwsSettings.json" />
    <EmbeddedResource Include="..\..\LocalApis.json" />
  </ItemGroup>
    6. Save the file.

    Configure the PetStoreMobileApp.Android Project

    1. Right Click on the "PetStoreMobileApp.Android Project". The context menu appears.
    2. Select "Properties". The properties tab opens.
    3. In the Application page, make sure the "Compile using Andriod Version:"" is set to "Android 10.0(Q).
    4. Ctrl-S to save the properties.

    Implement Pages

    The shell app we started with contains an About page and an Items List page. We are going to modify these pages and their ViewModels to use LazyStackAuth and PetStoreClientSDK classes.

    1. Download App.xmal.cs and move it into your PetStoreMobileApp project, replacing the one that is there.
    2. Download the following files and place them into your PetStoreMobileApp\Views folder, replacing the ones that are there:
      • AboutPage.xaml
      • AboutPage.xaml.cs
      • ItemsPage.xaml
      • ItemsPage.xaml.cs
    3. Download the following files and place them into your PetStoreMobileApp\ViewModels folder, replacing the ones that are there:
      • AboutViewModel.cs
      • ItemsViewModel.cs

    Run App

    1. Right Click on the PetStoreMobileApp.Andriod project. The context menu appears.
    2. Select "Set as Startup Project".
    3. Debug the project. The Android emulator opens and the App launches.
    4. Sign In
    5. Select Browse Items to see a list of Pets retrieved from the PetStoreDev stack.

    Note: We did not fully integrate the other pages in the sample app to use the PetStore stack. Only the Authentication page and Browse Items pages are currently making calls to the PetStoreDev stack.

    Step Summary

    In this step we started with a standard Xamarin Shell application and enhanced it to provide authentication services against AWS Cognito using the LazyStackAuth. In addition, we used the PetStore class from the PetStoreClientSDK library to make secure calls against the PetStoreDev stack.
